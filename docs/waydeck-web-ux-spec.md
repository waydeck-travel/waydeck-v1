## 1. Users & Goals

**Primary User:** Individual traveller planning trips for self + friends/family.

**Goals:**
- Create and manage trips with complete itineraries
- Store all travel documents (tickets, vouchers, visas) in one place
- Track expenses against budgets
- Reuse checklists and documents across trips
- Access trip info from any device (web + mobile)
